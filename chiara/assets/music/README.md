# Musica di Sottofondo

Metti qui il tuo file MP3 per la musica di sottofondo durante lo slideshow.

Il file deve chiamarsi: `background.mp3`

Puoi usare qualsiasi musica che ti piace, ma assicurati che:
- Il formato sia MP3
- Il file non sia troppo pesante (consigliato < 5MB)
- La musica sia adatta per un'atmosfera festosa

Se non vuoi usare musica, puoi rimuovere o commentare il tag `<audio>` in `index.html`.
