# Easter Egg Photo

This directory should contain a photo of Chiara for the easter egg feature.

## Instructions

1. Add a photo of Chiara to this directory
2. Name it `chiara-placeholder.jpg` (or update the filename in `index.html` if you use a different name)
3. The photo will appear when users click on "Chiara" in the homepage title

## Supported Formats

- JPG/JPEG
- PNG
- WebP

## Recommended Size

- Maximum width: 1200px
- Aspect ratio: Any (will be displayed responsively)
